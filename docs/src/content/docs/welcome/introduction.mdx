# Bienvenue sur FerrisKey

## À propos de FerrisKey

FerrisKey est une solution IAM (Identity and Access Management) conçue pour répondre aux enjeux **cloud-native** modernes : isolation multi-tenant, scalabilité dynamique et intégration transparente dans des architectures distribuées. Nous l’avons développé pour offrir :

- **Performance optimale** grâce à un cœur en Rust, garantissant vitesse d’exécution et faible empreinte mémoire.
- **Sécurité renforcée** par les garanties de mémoire de Rust et des pratiques éprouvées (cryptographie, gestion des secrets, JWT/OIDC).
- **Extensibilité et modularité** via une architecture hexagonale (ports & adapters) facilitant l’ajout de nouvelles fonctionnalités et l’intégration continue.

## Pourquoi Rust ?

Rust a été choisi pour :

1. **Sécurité mémoire** : élimination des erreurs courantes (use-after-free, data races) au moment de la compilation.
2. **Haute performance** : absence de garbage collector et optimisations de bas niveau.
3. **Concurrence sûre** : modèle ownership/borrowing qui simplifie l’écriture de code parallèle sans craindre les data races.

## Nos objectifs

1. **Simplicité d’utilisation** : interface Web intuitive et CLI ergonomique.
2. **Interopérabilité** : support natif de Helm, Kubernetes Operator, Terraform Provider.
3. **Transparence et open-source** : code ouvert, documentation exhaustive et communauté active.
4. **Évolution continue** : feedback utilisateurs orientant notre roadmap et nos releases régulières.
